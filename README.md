## 📝 1. GIỚI THIỆU ĐỀ TÀI
Dự án tập trung vào việc xây dựng một hệ thống quản lý toàn diện cho Studio ảnh, giúp tự động hóa quy trình đặt lịch (booking), quản lý sản phẩm ảnh chất lượng cao và tối ưu hóa việc phân công nhân sự (thợ chụp, thợ chỉnh sửa).

## 🎯 2. MỤC TIÊU DỰ ÁN
* **Số hóa quy trình:** Thay thế việc quản lý bằng sổ sách thủ công.
* **Tối ưu trải nghiệm:** Khách hàng có thể xem Portfolio và đặt lịch trực tuyến 24/7.
* **Quản lý dữ liệu:** Lưu trữ và bàn giao ảnh cho khách hàng một cách bảo mật, chuyên nghiệp.

## 🛠 3. CÔNG NGHỆ SỬ DỤNG
* **Frontend:** HTML5, CSS3, JavaScript (React.js/Vue.js)
* **Backend:** Node.js (Express) hoặc PHP (Laravel)
* **Database:** MySQL / PostgreSQL
* **Lưu trữ ảnh:** Cloudinary API (Tối ưu ảnh 4K/8K)
* **Thiết kế:** Figma

## 🚀 4. CÁC CHỨC NĂNG CHÍNH
### Dành cho Khách hàng:
* Xem Portfolio theo danh mục (Ảnh cưới, Ảnh nghệ thuật, Ảnh sự kiện).
* Đặt lịch chụp trực tuyến (Chọn ngày, giờ, thợ chụp).
* Nhận ảnh và tải ảnh chất lượng cao qua Gallery cá nhân.

### Dành cho Quản trị viên (Admin):
* Quản lý lịch hẹn qua giao diện Calendar.
* Quản lý danh sách nhân viên và thiết bị.
* Thống kê doanh thu và báo cáo tăng trưởng.
